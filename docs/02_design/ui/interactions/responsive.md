# レスポンシブ仕様

## ブレイクポイント定義

| 名称 | 幅 | Tailwind | 対象デバイス |
|------|---|---------|------------|
| sm | 640px | sm: | スマートフォン（横） |
| md | 768px | md: | タブレット |
| lg | 1024px | lg: | ノートPC |
| xl | 1280px | xl: | デスクトップ |
| 2xl | 1536px | 2xl: | 大型ディスプレイ |

## デバイス別レイアウト

### デスクトップ (xl: 1280px+)
- サイドバー: 常時表示（幅240px）
- メインコンテンツ: サイドバーの右側
- グラフ: 2列並び
- テーブル: フル幅表示
- フィルタ: 横一列に展開

### タブレット (md: 768px - 1279px)
- サイドバー: 折りたたみ（アイコンのみ、幅64px）
- メインコンテンツ: ほぼフル幅
- グラフ: 1列
- テーブル: フル幅（一部カラム非表示）
- フィルタ: 折り返し2列

### スマートフォン (〜767px)
- サイドバー: ハンバーガーメニュー（オーバーレイ）
- メインコンテンツ: フル幅
- グラフ: 縦積み（1列）
- テーブル: 横スクロール対応
- フィルタ: 縦積み + アコーディオン
- KPIカード: 2列 → 1列

## タッチ操作対応

| 操作 | デスクトップ | モバイル |
|------|------------|---------|
| ツールチップ | ホバーで表示 | タップで表示（再タップで非表示） |
| ドリルダウン | クリック | タップ |
| フィルタ | クリック | タップ |
| テーブルソート | ヘッダクリック | ヘッダタップ |
| 横スクロール | なし | スワイプ |

## パフォーマンス最適化

| デバイス | 最適化 |
|---------|--------|
| モバイル | 画像のサイズ最適化、グラフのデータポイント削減 |
| タブレット | 適度な情報量の表示 |
| デスクトップ | フル機能表示 |
