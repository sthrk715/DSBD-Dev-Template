# リリースチェックリスト

## ソースコード・ビルド (1-6)

- [ ] 1. Gitリポジトリにリリースタグが付与されている
- [ ] 2. CI環境でビルドが再現可能（package-lock.json完備）
- [ ] 3. 依存パッケージのバージョンが固定されている
- [ ] 4. リンター・フォーマッター警告が0件
- [ ] 5. 未使用コード・コメントアウトコードが除去済み
- [ ] 6. ハードコードされた認証情報・シークレットが存在しない

## テスト (7-12)

- [ ] 7. 単体テストカバレッジが基準値（80%）以上
- [ ] 8. 全自動テスト（Unit/Integration/E2E）がパス
- [ ] 9. テスト結果報告書が作成済み（エビデンス付き）
- [ ] 10. 性能テスト実施済み・要件充足
- [ ] 11. 受入テスト仕様書が準備済み
- [ ] 12. テストデータに本番個人情報が含まれていない

## セキュリティ (13-18)

- [ ] 13. SAST: Critical/High = 0
- [ ] 14. SCA: Critical/High = 0（依存パッケージの脆弱性なし）
- [ ] 15. 入力バリデーション・出力エスケープが適切
- [ ] 16. 認証・認可が要件通りテスト済み
- [ ] 17. TLS 1.2以上が適切に設定済み
- [ ] 18. ログに個人情報・機密情報が出力されていない

## ドキュメント (19-23)

- [ ] 19. 納品物一覧の全ドキュメントが揃っている
- [ ] 20. ドキュメントが最新のコード・構成と整合している
- [ ] 21. API仕様書が最新ビルド済み
- [ ] 22. 運用マニュアル・障害対応手順書が作成済み
- [ ] 23. 第三者が環境構築手順で環境再現可能

## ライセンス・コンプライアンス (24-26)

- [ ] 24. SBOMが生成済み・全ライセンス特定済み
- [ ] 25. GPL/AGPL利用時の法務確認完了
- [ ] 26. AI生成コード利用の開示・合意済み

## 運用準備 (27-30)

- [ ] 27. 監視設定（メトリクス、アラート閾値）が設定済み
- [ ] 28. バックアップ設定完了・リストア手順検証済み
- [ ] 29. CI/CDパイプラインが正常動作・本番デプロイ手順確立
- [ ] 30. インシデント対応体制（連絡先、エスカレーション）が整備済み

## 追加確認（GCP固有）

- [ ] Terraform applyの差分確認と実行
- [ ] 環境変数の設定確認（Secret Manager）
- [ ] Cloud Runリビジョンのトラフィック切替手順確認
- [ ] DNS / カスタムドメイン設定確認
- [ ] SSL証明書確認
- [ ] 本番データでの表示確認
