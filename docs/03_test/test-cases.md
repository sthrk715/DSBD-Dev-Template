## AI生成テストの品質チェック

- [ ] テストが有意義なアサーションを含んでいるか（トートロジーでないか）
- [ ] エッジケースが網羅されているか
- [ ] モックの設定が適切か（実装の詳細に依存していないか）
- [ ] テスト名が何を検証しているか明確か
- [ ] テスト間に依存関係がないか（独立して実行可能か）
