# プロジェクト概要

## 基本情報

| 項目 | 内容 |
|------|------|
| プロジェクト名 | [プロジェクト名] EC統合ダッシュボード |
| 目的 | 4つのECチャネル（Shopify自社EC、Amazon、楽天市場、Yahoo!ショッピング）の売上・顧客・サブスク・ギフト・メルマガ等のKPIを横断的に可視化し、データドリブンな意思決定を支援する |
| 背景 | 現状は各チャネルの管理画面を個別に確認しており、横断的なKPI把握やチャネル間比較に時間がかかっている。サブスクリプション・通年ギフト・eギフトなど複雑な売上構造の一元管理が必要 |

## 想定ユーザー

| ユーザー種別 | 説明 | 想定人数 |
|-------------|------|---------|
| Admin（管理者） | 全データ閲覧、フィルタ操作、データエクスポート、**ユーザー管理（招待・権限付与）**、システム設定 | 4-5名 |
| Viewer（閲覧者） | 全データ閲覧、フィルタ操作、データエクスポート | 10-20名 |

## 可視化対象

ダッシュボードで可視化するデータ領域:

1. **売上データ**: 全チャネル横断の日次/月次売上推移、チャネル別構成比、前年比/前々年比、目標達成率
2. **サブスクリプション**: 契約者数推移、コホート分析（月次リテンション）、解約率/スキップ率/一時停止率
3. **顧客分析**: 新規/リピーター売上比率、F2転換率、チャネル別客単価
4. **アクセス・CVR**: GA4セッション（検索/直接/広告/アプリ別）、商品ページ別CVR、流入元分析
5. **通年ギフト売上**: シーズン別（母の日/中元/敬老の日/歳暮/福箱）売上推移、eギフト内訳、前年比較
6. **商品カテゴリ別売上**: 注力商品追跡、カテゴリ構成比
7. **メルマガ分析**: 配信実績、開封率/クリック率推移、売上貢献
8. **時系列比較**: 前年比/前月比/前週比/同曜日比較、イベントフラグ付き推移、移動平均

|---------|------------|---------|-------------|
| Shopify（自社EC） | ~120,000件 | ~10,000件 | ~360,000件 |
| 楽天市場 | 74,040件 | 6,170件 | ~222,000件 |
| Amazon | 34,317件 | 2,860件 | ~103,000件 |
| Yahoo!ショッピング | 11,736件 | 978件 | ~35,000件 |
| **全チャネル合計** | **~240,000件** | **~20,000件** | **~720,000件** |

- Shopify顧客マスタ: 約60万件
- BigQuery推定ストレージ: ~5-8GB（無料枠10GB内）
